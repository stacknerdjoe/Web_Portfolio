# Joseph Onyenemerem — Web Portfolio

A clean and personal developer portfolio showcasing my projects, skills, and experience as a software developer based in Sweden.

🔗 **Live Site:** [josefportfolio.netlify.app](https://josefportfolio.netlify.app)

---

## ✨ Features

- 👋 **Personal Introduction** — A brief bio and quick links to email, GitHub, and LinkedIn
- 🎨 **Interactive Color Game** — A fun color picker that lets visitors customize the page background
- 🛠️ **Skills Table** — A structured overview of technologies organized by category
- 🗂️ **Latest Projects** — Showcases recent work with live links, GitHub repos, tech tags, and thumbnail images
- 📱 **Responsive Design** — Works across desktop and mobile devices

---

## 🛠️ Built With

- **HTML5** — Semantic markup and structure
- [FantaCSS](https://fantacss.smashing.com/) — Utility-based CSS framework for styling
- **Vanilla JavaScript** — For the interactive color game
- [Font Awesome](https://fontawesome.com/) — Icons for social links and UI elements

---

## 🚀 Getting Started

No build tools or dependencies required — this is a plain HTML project.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/stacknerdjoe/your-portfolio-repo.git
   ```

2. **Navigate into the project directory**
   ```bash
   cd your-portfolio-repo
   ```

3. **Open in your browser**

   Simply open `index.html` directly in your browser, or use a live server extension:
   ```bash
   # If you have VS Code Live Server installed:
   # Right-click index.html → "Open with Live Server"
   ```
